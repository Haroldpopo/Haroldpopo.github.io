# Write-up

提示是 web 代码修改上传验证，题目应该是浏览器端的上传检测漏洞，先右键查看源码

  1. 发现提交表单数据时有一个检测函数

    上传检测方案是在客户端使用 js 对不合法图片进行检查，很容易在前端对 js 脚本操作使其无效从而绕过

  2. 此为浏览器端的上传检测漏洞，在上传前把过滤函数 check() 删除即可实现检测绕过，直接把 return check() 删除就行

  3. 然后随便上传一个 php 文件，上传成功即出现 key
