# 360-SecLab-Passwd-03

# Brainfuck2

# 题目描述

Brainfuck 编码加密解密

题目网址：http://10.2.0.3/72/ok%EF%BC%9F.txt

# Write-up

  1. 进入题目地址，显示的全是 Ook ，根据提示去 Brainfuck 网址解密

    Brainfuck：https://www.splitbrain.org/services/ook

    粘贴进来，点击 Ook!_to_Text

  2. 解码得到一个类似 flag 形式的编码，这里去凯斯密码解密一下，偏移量一般是 13

    凯撒密码解密：http://www.atoolbox.net/Tool.php?Id=778

    解码得到 flag

# Brainfuck3

# 题目描述

Brainfuck 编码加密解密

题目网址：http://10.2.0.3/73/classic.txt

# Write-up

进入题目地址，显示了一堆符号，根据提示去 Brainfuck 网址解密

粘贴进来，点击 Brainfuck to Text

得到一个 JavaScript 脚本语句，在网页中执行一下这个语句，输出 flag

# rot13

# 题目描述

偏移量为 13 的凯撒密码

题目网址：http://10.2.0.3/74/

# Write-up

  1. 进入题目地址，得到一串编码

  2. 根据提示，找个凯撒密码解密的网址，解码得出 flag
